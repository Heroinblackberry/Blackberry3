В начало

Результатом работы с регулярным выражением может быть: проверка наличия искомого образца в заданном тексте; определение подстроки текста, которая сопоставляется образцу; определение групп символов, соответствующих отдельным частям образца. Если регулярное выражение используется для замены текста, то результатом работы будет новая текстовая строка, представляющая из себя исходный текст, из которого удалены найденные подстроки (сопоставленные образцу), а вместо них подставлены строки замены (возможно, модифицированные запомненными при разборе группами символов из исходного текста). Частным случаем модификации текста является удаление всех вхождений найденного образца — для чего строка замены указывается пустой.
